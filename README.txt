
SETUP IN TERMUX

1. Install packages

pkg update -y
pkg install python -y

2. Install requirements

pip install -r requirements.txt

3. Run website

python app.py

4. Open browser

http://127.0.0.1:5000

FEATURES

- Real bash terminal
- Ubuntu style UI
- Supports:
  ls
  cd
  pwd
  python
  pip
  pkg
  git
  curl
  wget
  nano
  vim

No Docker required.
